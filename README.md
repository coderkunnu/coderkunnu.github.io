## Basic GIT Commands

Initilise a new local git repository
```markdown
git init
```
Link a local git repository to your github repository
```markdown
git remote add origin <repository_url>
```

Clone a repository
```markdown
git clone <repository_url>
```

Get latest changes
```markdown
git fetch
```

Switch Branch
```markdown
git checkout <branch>
```

Pull latest changes into your code
```markdown
git pull
```

Look for changes done
```markdown
git status
```

Add all files for staging
```markdown
git add --all
```

Commit the code with commit message
```markdown
git commit -m "commit message"
```

Push to the repository
```markdown
git push
```
